# Webserver



Implemented using flask.



To run:

```bash
cd login_server
python app.py
```

It runs on http://127.0.0.1:5000/





~~[deprecated] To run:~~

```bash
# In the parent directory
export FLASK_APP=login_server
export FLASK_ENV=development

flask run
```

~~It runs on http://127.0.0.1:5000/~~





Requirements:

```bash
sudo apt-get install libmysqlclient-dev
pip install flask_mysqldb
pip install flask
```
